# 织影

为钩织方便专门做的离线计数工具，
浏览器打开即用，可安装到桌面。

# 功能

- 多项目切换 — 同时管理多个编织项目，进度互不干扰
- 部件分组计圈 — 按袖子、前片、后片等自定义分部件计数，每圈标记完成状态
- 文字图解导入 — 粘贴或拍照识别图解，自动解析针法符号
- 三种操作模式 — 计数模式、编辑模式、跳转模式，一键切换
- 离线可用 — PWA 安装后断网也能用，数据存在于手机本地

# 使用

1. 点击「新建项目」，输入名称创建第一个项目
2. 添加部件（如左袖、前片），按图解逐圈点击计数
3. 右上角可切换项目、导出 PDF 或备份数据
